Entendendo o Server na Web

Basicamente o conceito de servidor, é onde armazena todos os arquivos e que pode acessar via lan, e os servidores tendem a fazer comunicações entre outros computadores, e muitas das vezes podem armazenar arquivos importantes.

Existem diversos tipos de servidores, que executam algum serviço dentro do servidor e é passado pelo cliente via conexão, dentre eles:

    Alguns exemplos
  • Streaming
  • Segurança
  • Email
  • Servidor WEB

    • Existem empresas dedicadas ao fornecimento de servidores para clientes que não possuem uma infraestrutura adequada para construir um servidor

    Esssas empresas são chamadas de DATACENTERS

    Basicamente são um conjunto de servidores, computadores enormes que servem para aguentar acessibilidade de usuários (Como o próprio Facebook e Google que possui um datacenter próprio devido a seu número grande de usuários.)

    Servers SOFTWARES e HARDWARES

    HARDWARES

    Tem o computador físico, onde é armazenado o DISCO RÍGiDO, seja um HD SATA, ou SSD, memória ram.

    Ou qualquer outra coisa do tipo que seja físco, trabalhando em operação para o funcionamento dos SOFTWARE.

    SOFTWARES

    Existem o SOFTWARES como os sistemas operacionais, feitos e desenvolvido para gerenciar própriamente os computadores de servidores.

      Como exemplo, temos alguns exemplos de sistemas operacionais
  • Windows Server
  • CentOS Linux
  • Oracle Linux
  • Oracle Debian
      • Temos os servidores WEB
  • APACHE HTTP PROTECT
  • NGINX

    • E como funcionam esses servidores WEB?

    Basicamente, o servidor WEB funciona através do Requisito HTTP, que interpreta e conversa com o servidor, afim dele converter e responder de volta com HTML, CSS, Javascript imagens ou texto.
