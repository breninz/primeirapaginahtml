Cliente na visão WEB

Basicamente o cliente na visão WEB entende-se, que o Cliente acessa o site através do Protocolo HTTP fazendo com que o mesmo acesse o site atráves de um servidor Host que hospeda o site. Com a internet fazendo uma intermediação entre as duas conexões.


Navegadores

Também chamado de Browsers é responsável pela intermediação e tradução para o Cliente

Os navegadores interpretam e traduzem o HTML, CSS e Javascript, virando textos, audios e imagens para o cliente.

Aplicações Web

São aqueles aplicativos que acessam a internet atráves do APP sem precisar de um navegador instalado para utilizar, sendo feito a partir próprio aplicativo

são diferentes de qualquer aplicativo comum feito a partir de Python ou qualquer outra linguagem por exemplo.
